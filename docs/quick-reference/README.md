# Quick Reference Guide

This section provides at-a-glance overviews of all 66 books of the Bible in easy-to-scan table format.

## Available Quick References

- **[Old Testament Overview](old-testament-overview.md)** - All 39 OT books in one table
- **[New Testament Overview](new-testament-overview.md)** - All 27 NT books in one table

## How to Use These Tables

Each table includes:
- **Book Name** - Links to the full study guide for that book
- **One-Sentence Summary** - Ultra-concise overview (20-40 words)
- **Key Theme** - The book's central focus
- **Key Verse** - Most important reference
- **Top Application** - One practical takeaway

## Tips for Printing

These tables are designed to fit on standard letter-size paper:
- Use landscape orientation for best results
- Adjust margins to 0.5" if needed
- Consider printing double-sided (OT on one side, NT on the other)

## Best Uses

1. **Quick Lookup** - Find the essence of any book in seconds
2. **Teaching Prep** - Get overview before diving into full study
3. **Bible Reading Plans** - Track progress and remember key points
4. **Group Study** - Distribute as handouts for class overviews
5. **Memory Work** - Review summaries to internalize Bible structure
